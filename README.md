# nestjs-starter
[![Build Status](https://travis-ci.org/UtoYuri/nestjs-starter.svg?branch=master)](https://travis-ci.org/UtoYuri/nestjs-starter)

A tool for creating `Nest` project.

## Usage
### Dev Mode
```
yarn start:dev
```
### Production Mode
```
yarn prestart:prod && yarn start:prod
```
### Lint
```
yarn lint
```
### Test
```
yarn test
yarn test:watch
yarn test:cov
```
